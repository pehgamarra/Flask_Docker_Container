Esta aplicação foi desenvolvida para demonstrar meu conhecimento em containers Docker, utilizando o framework Flask para criar uma API simples. O projeto exibe um JSON com a mensagem "hello world", servindo como uma base para futuras expansões e integrações.

Principais Recursos:
Containerização com Docker: Aprendi a criar, gerenciar e executar containers Docker, permitindo que a aplicação seja facilmente configurada e escalável em qualquer ambiente.
API Flask: Utilizei o Flask para criar uma API leve, que pode ser facilmente expandida com novas rotas e funcionalidades.
Estrutura Modular: A aplicação é estruturada de forma modular, facilitando a manutenção e a adição de novas features no futuro.
Este projeto é uma demonstração prática de como implementar aplicações modernas utilizando Docker, mostrando minha habilidade em containerização e desenvolvimento web.
